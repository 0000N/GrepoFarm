# 🌾 GrepoFarm

Farm villages automatique pour Grepolis. Simple, rapide, efficace.

## Installation

1. Installe [Tampermonkey](https://www.tampermonkey.net/)
2. [Clique ici pour installer](https://raw.githubusercontent.com/0000N/GrepoFarm/main/grepofarm.user.js)
3. Lance Grepolis → le panel apparaît en haut à droite

## Utilisation

- **Clique sur le titre** pour activer/désactiver
- **Choisis le timing** : 5, 10, 15, 20, 30 ou 45 minutes
- Le chrono indique la prochaine collecte

Le bot choisit automatiquement la ville la plus pauvre par île.
